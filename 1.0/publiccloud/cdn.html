HP Helion Public Cloud CDN

Using the HP Helion Public Cloud Content Delivery Network (CDN) service, content inside HP Helion Public Cloud Object Storage can easily be cached on servers in metropolitan areas all over the world to speed client access to them by removing much of the network latency. Once a container created with Object Storage has been enabled with the CDN service, all objects within that container are automatically distributed to servers on the worldwide Content Delivery Network.

For a step-by-step guide to enable CDN on the HP Helion Public Cloud Console, take a look at the Knowledge Base article Activating CDN for your Object Storage.

If you have any questions, try our Forums where you can learn from our own internal experts as well as other users in the HP Helion Public Cloud community.

API

Do you need low level, raw REST API access to HP Helion Public Cloud CDN? Head on over to the CDN API page for details, including a video showing usage of the API using cURL.
