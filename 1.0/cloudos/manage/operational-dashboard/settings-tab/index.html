Operational Dashboard: Settings Tab

The Settings tab contains the following panels:

  • User Settings

  • License

  • About

For procedural instructions, see Install and Configure Your Clouds.

User Settings

This topic provides details about the User Settings panel. The following table describes the User Settings UI elements:

UI Elements Description
Language Defaults to English (en).
Timezone Defaults to UTC (Coordinated Universal Time). To change the default, select the timezone for your area from the drop-down list.
Table page size Defaults to 10 - a maximum of 10 line items can be displayed at one time. To change the default, enter a different number.
Log level Defaults to Info. The five levels used for message logging are, in order of gravity: Critical, Error, Warning, Info, and Debug.

License

This topic provides details about the License panel.

On the License page, you can view the number of days remaining on your current HP Cloud OS license, and update the license if necessary. The following table describes the License UI elements:

UI Elements Description
Number of days remaining on current license The number of days currently remaining on the HP Cloud OS license. This field will show zero (0) when the license is expired. If the license is expired, you cannot use the HP Cloud OS dashboards until you renew the license.
Instance ID A unique alphanumeric string that identifies the particular installation of HP Cloud OS to which the license applies. You must provide this identifier to HP Support in order to obtain a valid license key, and the license key will be valid only for the HP Cloud OS installation having this Instance ID.
License key Enter a valid license key into this field and click the Save button to update the HP Cloud OS license. Contact your HP Support representative to obtain a valid license key.

Note: After you click the Save button, about 20 minutes elapses before the new license is activated.
Log level Defaults to Info. The five levels used for message logging are, in order of gravity: Critical, Error, Warning, Info, and Debug.

About

The About panel displays the version of the Operational Dashboard and the HP Cloud OS build, and the copyrights.
