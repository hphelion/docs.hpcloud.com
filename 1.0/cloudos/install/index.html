Install & Configure Your Cloud

Welcome! The following topics explain how to install & configure your cloud with the HP Cloud OS platform.

It's important to read the topics and perform the described steps in order. While some steps are optional, the sequence that you'll follow in the HP Cloud OS Operational Dashboard user interface is significant, to ensure a proper deployment of your cloud.

Install and set up the Admin node:

  • Installation Overview
  • Before You Install
  • Set Admin Node Prerequisites (Required)
  • Customize Server Types (Optional)
  • Customize Network Connections (Optional)
  • Customize Network Settings (Required)
  • Complete Admin Node Installation (Required)

After you install and set up the Admin node, you can perform additional steps:

  • Create a Cloud
  • Launch Administration Dashboard
  • Check the HP Cloud OS License
  • Install Hot Fixes
  • Updates and Extensions

Finally, if you do not intend to use the wizard-based Cloud OS installation process that's described in the Create a Cloud topic, and instead decide to apply and configure the install modules yourself, see these topics:

  • Advanced Cloud Setup
  • Install Modules Reference

Note: Also see the Help topics that are available in the Operational Dashboard and Administration Dashboard. Website copies are available:

  • HP Cloud OS Operational Dashboard Help
  • HP Cloud OS Administration Dashboard Help
