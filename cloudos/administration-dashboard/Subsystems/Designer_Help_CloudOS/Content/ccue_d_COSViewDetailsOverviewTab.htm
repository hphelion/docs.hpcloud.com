View Design Details

  1. On the Project tab, select a Current Project.
  2. Click Provisioning > Topology Designs to open the Browse Designs window.
  3. Select a design from the list of Deployment Templates. See Browse Designs for more information on how to filter the list of designs.
  1. A view-only graphical representation of the design components opens in the right panel of the Overview tab with the following details displayed on the left:
  • Name - The name of the design.
  • Last Modified - The date, time, and user who last updated the design.
  • Cloud Profile - Represents the supported OpenStack version and capabilities of the cloud to which you are deploying.
  • Resource Pool - The Resource Pool selected during creation.
  • Validation status - Indicates the validation status of the Document, Nodes, Relations, and Profiles.
  • Structure Overview - Shows the number of nodes and connections.
