How to Create a Router

You can create a router to interconnect subnets and forward traffic within the subnets. A router also enables you to NAT internal traffic to external networks.

A router has an interface for each subnet it is associated with. By default, the IP address of this interface is the subnet's gateway IP. When a router is associated with a subnet, a port for that router interface will be added to the subnet's network.

 

To create a router:

  1. On the Project tab, select a Current Project.

  2. Click Infrastructure > Routers to open the Routers window.

  3. Click Create Router.

  4. On the Create Router dialog, enter a Router Name.

  5. Click Create Router.
