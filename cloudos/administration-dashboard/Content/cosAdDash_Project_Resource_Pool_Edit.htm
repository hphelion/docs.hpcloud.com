How to Edit a Resource Pool

Use the Edit Resource Pool view to update resource pool definitions that will be used during topology provisioning. In common use cases, you can:

  • Change the name of the resource pool.
  • Switch to a new region.
  • Update the OpenStack server domain URL.
  • Modify different provider credentials when the resource is an HP Cloud.
  • Modify values that were not accurate when you originally entered them.

Note: If you change your HP Cloud password, you must also change your password in the Edit Resource Pool view.

 

To update resource pool definitions:

  1. On the Project tab, select a Current Project.

  2. Click Provisioning > Resource Pools to open the Resource Pools window.

  3. Find the resource pool that you want to edit, and click Edit Resource Pool.

  4. In the Details section, modify the Name, Type, Region ID, Version, Scope, Cloud Profile, and Provider URL.

  5. Depending on the Cloud Type, in the Provider Credentials section, modify the Tenant ID, User Name, and Password. Enter the password again in Confirm Password to validate. If you create a resource pool that requires credentials, the Administration Dashboard validates them when you edit a resource pool, instead of when you first use the resource pool.

  6. Click Update Resource Pool to save your changes.
