Before You Begin with the Administration Dashboard Help

Before you begin with the HP Cloud OS Administration Dashboard Help, take a few minutes to understand how the help content is organized. You can find information in this order:

  • Conceptual introduction and prerequisites. See Welcome to the HP Cloud OS Administration Dashboard.

  • A Getting Started section that provides instructions for how to begin using the dashboard. You can read about responsibilities for each user role, and follow links to step-by-step instructions for performing tasks. See Getting Started with the Administration Dashboard.

  • A section with step-by-step instructions for working with deployment profiles and topology designs. See Topology Designs.

  • A section that lists tasks for each role. (This is a comprehensive list, unlike Getting Started.) See Administration Dashboard Tasks by Role.

  • A section that provides step-by-step instructions for tasks. This section's organization is based on the user interface tabs and windows. See Administration Dashboard Tasks by Tab.
