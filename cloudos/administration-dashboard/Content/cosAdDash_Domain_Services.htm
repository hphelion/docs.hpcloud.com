Domain Services

Keystone is the security context and the defining service for a cloud. All services connected to a single Keystone service belong in one cloud. Glance, in the recommend configuration, is also a cloud service because the images it provides are available to VMs in all regions.

OpenStack cloud services are Cinder, Glance, Keystone, and Quantum. A cloud service is a service that can be used by or for any region, so all of the HP Cloud OS services are considered cloud services because they can be used for one or more regions.

How to Use the REST API Application

The HP Cloud OS API Documentation application is a tool for describing, producing, and visualizing RESTful web services. Its purpose is to enable client and documentation systems to update at the same pace as the server. The documentation of methods, parameters, and models are tightly integrated into the server code, allowing APIs to stay in sync.

If a service supports the HP Cloud OS API Documentation application, it displays a REST API button.

Click REST API to start the documentation application for a service.

For information about how to use the documentation application, see http://docs.hpcloud.com/cloudos/120/en/quickstart/. (IE will update the link when we have the real location.)
