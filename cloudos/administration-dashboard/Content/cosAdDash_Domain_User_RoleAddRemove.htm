How to Add or Remove a Role from a User

Note: You must be logged in to the Administration Dashboard as a cloud administrator.

After you add a user to a project, you can add additional project roles to the user.

Update

To add a project role to a user:

  1. Select Projects to open the Projects view.

  2. Find the user's project in the list, and click Modify Users to open a new view.

  3. Find the user (in the Users In (Project Name) list), and click Add Project Role to open a dialog box.

  4. Select the project role and click Add Role.

 

To remove a project role from a user:

  1. Select Projects to open the Projects view.
  2. Find the user's project in the list, and click Modify Users to open a new window.
  3. Find the user (in the Users In Project Name list). Click More, and select Remove Project Role.
  4. In the dialog box that opens, select the role to remove and click Remove Role.
