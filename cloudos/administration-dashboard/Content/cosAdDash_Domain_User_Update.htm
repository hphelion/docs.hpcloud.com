How to Edit a User

Cloud administrators can edit users.

Note: You cannot edit control users such as Trash or Nova. Edit does not display for users that cannot be edited.

 

To edit a user:

  1. Click Domain > Users to open the Users window.

  2. Find the user to edit, and click More > Edit to open the Update User dialog box.

  3. On the User tab of the dialog box, update the User Name, Email, or Password.

  4. On the Cloud tab, update the user's cloud role. See Administration Dashboard Roles.

  5. Click Update User.
