How to Create a Subnet

You can create a subnet on a network. Subnets allow you to divide a network into addressable segments.

Note: You must be the Admin user or a cloud administrator to create a subnet on an external network or shared network.

 

To create a subnet:

  1. On the Project tab, select a Current Project.

  2. Click Infrastructure > Networks to open the Networks window.

  3. Find the network where you want to add a subnet, and click More > Add Subnet to open the Create Subnet dialog box.

    Note: An alternate method is to click Create Subnet on the Networks Overview window. The Create Subnet dialog box opens.

  4. On the Subnet tab of the dialog box:

    1. Enter a unique Subnet Name.

    2. Enter a Network Address from your assigned, dedicated IP address range.

    3. Keep the defaults for IP Version, Gateway IP, and Disable Gateway.

  5. On the Subnet Detail tab of the dialog box:

    1. Leave Enable DHCP.

    2. Enter an Allocation Pools IP address list for the subnet, using one line in the entry box for each address.

    3. Enter a DNS Name Servers IP address list for the subnet, using one line in the entry box for each address.

    4. Enter a Host Routes IP address list to announce additional routes to the hosts. Use one line in the entry box for each address.

  6. The Networks window opens, and you see the subnet.
