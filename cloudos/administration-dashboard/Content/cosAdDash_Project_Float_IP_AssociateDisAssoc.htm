How to Associate and Disassociate Floating IPs

You can associate and disassociate floating IPs.

 

You can assign an address to a particular VM instance.

To associate a floating IP:

  1. On the Project tab, select a Current Project.

  2. Click Access and Security > Access and Security > Floating IPs tab.

  3. In the Floating IPs list, select an IP address and click Allocate IP to Project.

  4. In the Allocate Floating IP dialog, select a Pool from the drop-down list.

  5. Click Allocate IP.

  6. Wait for the success message.

 

You can make a floating IP available for association within the same project. Releasing a Floating IP releases to it the pool and makes it available to all projects.

To disassociate a floating IP:

  1. On the Project tab, select a Current Project.

  2. Click Access & Security > Access & Security > Floating IPs tab.

  3. In the Floating IPs list, select an address where the Disassociate Floating IP action is available.

  4. Click Disassociate Floating IP.

  5. Wait for the success message.
