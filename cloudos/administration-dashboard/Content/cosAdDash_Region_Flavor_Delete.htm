How to Delete a Flavor

Only a domain administrator within a region can delete flavors. When you delete a flavor, the flavor is no longer visible in the Administration Dashboard and cannot be used to create a new VM, to assign in resource pool services, or to specify in Infrastructure Design documents.

A deleted flavor still exists in the Nova database until the clean flavors script (scheduled by a system administrator) runs. The script cannot delete a flavor, if the flavor is in use by an active instance. You must terminate the instance.

Note: If all flavors are deleted, you cannot create VMs.

 

To delete one or more region flavors:

  1. On the Region tab, select a Current Compute Region.

  2. Click Flavors to open the Flavors window.

  3. Select the flavors you want to delete. To select all, click the top checkbox in the header row.

  4. Click Delete Flavors.

  5. In the Confirm Delete Flavors dialog box, verify your selections and then click Delete Flavors.

  6. Wait for the success message.
