How to Create a Volume

You can create a volume by entering the size, or you can use a volume snapshot as a source. After you create a volume, you can attach it to a VM and place content in it.

 

To create a volume:

  1. On the Project tab, select a Current Project.

  2. Click Infrastructure > Volumes to open the Volumes window.

  3. Click Create Volume to open the Create Volume window. Under the Volume Quotas heading, review the:

    1. available disk space

    2. number of volumes allocated

    3. number of volumes available

      Note: If you are logged in as an Administrator, Nova returns all volumes. Volume Quotas might show that no volumes are available to create. In this case, log out and log back in as a User rather than as an Administrator.

  4. To create a new volume that is not based on a snapshot, enter:

    Volume Name

    Description (Optional)

    Type — Defined by cloud administrators, at the region level.

    Size

  5. To use a snapshot as a source for the new volume, select the snapshot from the field drop-down and it auto-populates the Volume Name, Type and Size. Enter an optional Description.

  6. Click Create Volume. The volume is created and available to attach to VMs.
