How to Attach or Detach Volumes to an Instance

Volumes are disk storage volumes that have been created in a project and can be attached to an instance. A volume can only be attached to a single VM at a time, and that VM has sole use of that volume.

 

To attach a project volume to an instance:

  1. On the Project tab, select a Current Project.

  2. Click Infrastructure > Volumes to open the Volumes window.

  3. Find the volume you want to attach, and click Edit Attachments to open the Manage Volume Attachments dialog box.

  4. Select an instance from Attach to Instance.

  5. Enter a Device Name, which is an attached device, such as a device between /dev/vdc and /dev/vdz.

  6. Click Attach Volume.

 

To detach a volume from an instance:

  • After a volume is attached, Detach Volume is available through the Attach Volume action.
