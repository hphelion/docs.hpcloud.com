How to Change User Settings

To change Administration Dashboard display settings:

  1. Click Settings in the upper right of any window.

  2. Click User.

  3. In the User Settings window, keep or change the following settings:

    1. Language: English (en).

    2. Timezone: UTC (Coordinated Universal Time). To change the default, select a timezone from the drop-down list.

    3. Items Per Page: This number represents how many line items display on one page. To change the default, enter a different number.

  4. Click Save.

  5. Wait for the success message.
