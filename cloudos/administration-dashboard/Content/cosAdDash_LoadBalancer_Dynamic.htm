How to Create a Dynamic Load Balancer

You can create a dynamic load balancer that provisions a load balancer instance when you provision a topology. When you use this load balancer method, you create a load balancer instance for each topology.

 

To create dynamic load balancers:

  1. Create an image, if it does not already exist. See How to Create an Image.

  2. Create a resource pool with a load balancer service that specifies the load balancer image. See How to Create a Resource Pool and How to Add or Edit a Resource Pool Service.

  3. Use a resource pool with a load balancer service and include a load balancer in your design. See Create a Profile.

  4. When you provision the topology, a load balancer instance will be provisioned. See How to Launch a Deployment Profile.
