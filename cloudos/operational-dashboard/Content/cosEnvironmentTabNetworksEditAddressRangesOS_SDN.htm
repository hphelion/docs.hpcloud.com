OS_SDN Network Edit Address Ranges Dialog

To access: Select the Environment tab > Networks > click the down arrow and select Edit Address Ranges for the os_sdn network type.

Caution: If you plan to customize the address ranges, you must do that before you complete the steps on the Complete Install panel- see Complete Install. After that, the Edit Address Ranges dialog will be read-only. Once the install process is triggered, no changes can be made. If you want to make changes, you need to start over by re-installing the HP Cloud OS Administration node. See the Install section of the HP Cloud OS Documentation web site for instructions.

The following table describes the os_sdn network type address range default UI elements:

UI Elements Description
Network Type os_sdn
Subnet Pre-populated with the correct setting: 192.168.130.0
Node Types host

IPV4 Start Addr /
IPV4 End Addr

Permanent admin network IP address assignment for the participating nodes -- 192.168.130.81 to 192.168.130.160
Update Address Ranges button

Click to save the edited settings, and to validate that the IP address ranges are correct.

Note: If you enter invalid ranges, you will get a validation error when you click Update Address Ranges. The validation error identifies the incorrect address (start or end). You must fix all errors in this Edit Address Ranges dialog for the install to be successful. Clicking Cancel will not trigger the validation step. identify the incorrect address.
